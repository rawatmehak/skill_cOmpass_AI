# 🚀 SkillCompass AI

SkillCompass AI is an intelligent career guidance platform that helps users discover the most suitable tech career paths based on their skills and resume analysis.

It combines **AI-powered insights**, **skill analysis**, and **real-time recommendations** to guide users toward better career decisions.

---

## 🌟 Features

* 📄 **Resume Upload & Analysis**
  Upload your resume and let AI extract relevant skills automatically.

* 🧠 **Skill-Based Career Recommendation**
  Get personalized job role suggestions based on your skills.

* 📊 **Skill Gap Analysis**
  Identify missing skills required for your target role.

* 📈 **Career Readiness Score**
  See how prepared you are for a specific career path.

* 🛠 **Manual Skill Input**
  Add and rate your skills manually for better accuracy.

* 🔐 **Authentication System**
  Secure login and registration for users.

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/SkillCompass_AI.git
cd SkillCompass_AI
```

---

### 2️⃣ Setup Backend

```bash
cd backend
npm install
npm start
```

---

### 3️⃣ Setup Frontend

```bash
cd skillcompass-frontend
npm install
npm run dev
```

---

## 🔐 Environment Variables

Create a `.env` file in the backend folder:

```
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
```

---

## 🚀 How It Works

1. User uploads resume or enters skills manually
2. System extracts and analyzes skills
3. Backend processes data and calculates:

   * Best matching role
   * Match percentage
   * Missing skills
4. User gets a **personalized career roadmap**
